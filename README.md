# 💖 Valentine's Day Interactive Website

A fun and interactive Valentine's Day themed mini web project built as my first website.

## 🚀 About The Project

This project is a playful interactive website where the "No" button runs away when the user tries to click it. Eventually, the user is encouraged to click the "Yes" button.

This project helped me understand:
- Basic HTML structure
- CSS styling and positioning
- JavaScript DOM manipulation
- Mouse events and interactivity

## 🎯 Special Feature

✨ Interactive "No" Button  
When the mouse cursor moves near the "No" button, it changes position dynamically, making it difficult to click.

- JavaScript (Vanilla JS)
